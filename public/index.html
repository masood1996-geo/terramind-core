Help — Satellite Data & Analysis

🔥 Fire / Burn Scars
NBR = (NIR − SWIR) / (NIR + SWIR)
Sentinel-2: (B8A−B12)/(B8A+B12) | Landsat: (B5−B7)/(B5+B7). Use dNBR = pre−post for severity. SWIR False Color: B12-B8A-B4.
🌍 Earthquake / Deformation
InSAR — Sentinel-1 SLC pairs
Phase difference pre/post quake → ground displacement. Coherence loss = building damage. Software: ESA SNAP, ISCE2. ASF DAAC for free SAR data.
🌊 Flood Mapping
NDWI = (Green − NIR) / (Green + NIR)
Sentinel-2: (B3−B8)/(B3+B8). SAR: Sentinel-1 VV < -15dB = water (works through clouds). Mask permanent water with JRC dataset.
🌋 Volcanic Activity
TIR + SO₂ Column (TROPOMI)
Landsat B10 for thermal anomalies. Sentinel-5P for SO₂ plumes. Sentinel-1 InSAR for magma deformation. MIROVA for real-time thermal alerts.
⛈️ Storm Damage
SAR Coherence + NDVI Change
Sentinel-1 pre/post coherence loss = structural damage. NDVI drop from vegetation stripping. GPM/IMERG for precipitation data.
